# SeferFlow Plugin
**Status**: Minimal working plugin (v1.0.0)

## Installation

1. In Obsidian Settings → Community Plugins
2. Click "Browse" tab
3. Search "seferflow"
4. Click "Enable"
5. Restart Obsidian

## Features

- 💬 Simple status panel
- 🔐 Login/logout button
- 📊 Usage tracking
- ✅ No dependencies

## Known Issues
